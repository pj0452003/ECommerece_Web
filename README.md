**Project Overview**
This e-commerce website allows users to browse products, add items to a cart, and proceed to checkout. The website is responsive and designed for seamless performance across devices.

**Key Highlights:**
Clean and intuitive UI for product browsing and checkout.
Real-time cart updates.
Modular and scalable architecture for easy future enhancements.

**Setup Instructions**

1.Clone the repository:git clone https://github.com/pj0452003/ECommerece_Web.git  

2.Navigate to the project directory: cd ECommerece_Web  

3. Install dependencies:npm install (and you can also ensure to you installed vite and Tailwind Css )

4. Start the development server: npm run dev

**Running the Project**
1.Ensure all dependencies are installed.
2.Start the Vite development server: npm run dev  
Open your browser and navigate to http://localhost:5173/ to view the project.

**Technologies Used :**
1.Frontend: React.js, HTML/CSS/jsx
2.Build Tool: Vite
3.Styling: Tailwind CSS
